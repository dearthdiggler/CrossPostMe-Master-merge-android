# API Endpoints for iOS App

Reference endpoints from your backend:

- POST /api/auth/register
- POST /api/auth/login
- GET /api/status
- POST /api/status
- GET /api/health
- GET /api/ready

## Chat, Email, and WhatsApp Integration Endpoints

- POST /api/chat/send
- GET /api/chat/history
- POST /api/email/send
- POST /api/whatsapp/send

Use these endpoints in your iOS app to interact with the backend agent and services.
